# `@christiannaths/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @christiannaths/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@christiannaths/prettier-config"
}
```

or

**Edit `.prettierrc.json`**:

```jsonc
"@christiannaths/prettier-config"
```
